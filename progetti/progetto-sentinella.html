Descrizione del Progetto

Questo progetto mira a replicare una "modalità sentinella" per qualsiasi veicolo. Utilizzando un microcontrollore come Raspberry Pi e un sensore a ultrasuoni, il sistema è in grado di rilevare oggetti o persone che si avvicinano troppo all'auto. Quando viene rilevata una presenza, il sistema può attivare una serie di azioni, come accendere le luci, far partire una registrazione video o inviare una notifica allo smartphone del proprietario.

Componenti Necessari

Componente Quantità Note
Raspberry Pi 4 1 Qualsiasi modello con GPIO va bene.
Sensore a Ultrasuoni HC-SR04 4 Per coprire tutti i lati del veicolo.
Power Bank 1 Con capacità di almeno 10,000mAh.
Cavi Jumper Set Maschio-Femmina e Maschio-Maschio.

Schema del Progetto

Di seguito è riportato lo schema elettrico per il collegamento di un singolo sensore a ultrasuoni al Raspberry Pi. Il processo va ripetuto per ogni sensore, utilizzando pin GPIO differenti.

Implementazione

L'implementazione software è gestita da uno script Python. Per prima cosa, assicuriamoci di avere le librerie necessarie installate sul nostro Raspberry Pi. Apriamo il terminale e digitiamo: 

sudo apt-get update
sudo apt-get install python3-pip
pip3 install RPi.GPIO

Una volta installate le dipendenze, possiamo iniziare a scrivere il codice per leggere i dati dal sensore. Creiamo un file chiamato `sentry.py` e inseriamo il codice iniziale per la configurazione dei pin GPIO...
